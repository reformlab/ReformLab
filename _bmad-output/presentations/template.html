Every assessment starts from scratch

1
Find and prepare data
Population surveys, energy consumption, emission factors — scattered across sources, incompatible formats
2
Build the simulation
Custom scripts connecting income, energy, and behavioral models — rebuilt from scratch every time
3
Run projections
Manually loop year by year — track fleet turnover, heating transitions, policy phase-in
4
Analyze impact and optimize
Custom code for distributional impact, fiscal cost, subsidy calibration, rebate optimization
5
Document methodology
Usually skipped entirely. No audit trail, no reproducibility.
The result
Weeks
Unreliable data cobbled from fragile sources
Spaghetti code nobody else can run
Excel files passed around as "models"
Undocumented assumptions baked in
When the minister asks "what about low-income rural households?" — you start over.
