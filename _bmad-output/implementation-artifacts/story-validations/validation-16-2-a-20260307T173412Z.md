**Impact:** Reproduction can be impossible in “clean environment,” enabling misleading pass/fail outcomes.

**Recommended Fix:**

**Impact:** False positives/false negatives in reproducibility verdict; runtime failures on valid datasets.  
**Source:** Task 3.5 and `_compare_panel_tables()` algorithm.

**Problem:**  
Comparison hardcodes sorting by `(household_id, year)` and leaves null/NaN/tolerance semantics undefined.

**Recommended Fix:**  
Define and enforce:
1. Explicit comparison keys source (manifest metadata).  
2. Null/NaN and non-finite value behavior.  
3. Numeric type coercion and tolerance semantics.  
4. Fallback path when required keys are absent.
