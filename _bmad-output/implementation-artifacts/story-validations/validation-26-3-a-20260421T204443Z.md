### 1. Runtime Mode Type System Mismatch

**Impact:** Implementation will fail - the specified approach cannot work as written
**Source:** Story "Runtime Summary Section" and code sample for `lastRuntimeMode`

**Problem:**
The story instructs developers to find runtime mode via:
```typescript
const lastRuntimeMode = useMemo(() => {
```

**Recommended Fix:**

Option B: Fetch full `ResultDetailResponse` for the last run (requires API call)
Option C: Use `AppContext.runResult.runtime_mode` but this is transient (lost on refresh)

Document which approach to use and update the code sample accordingly.
