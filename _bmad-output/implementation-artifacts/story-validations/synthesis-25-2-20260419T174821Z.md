- **"Removal ambiguity - how do users unadd/undo?"** | **Raised by**: Validator B | **Dismissal Reason**: The composition panel has remove buttons (existing functionality). Users remove instances from composition, not from browser. Browser is catalog-only with add action.

- **"Mobile AC not testable via RTL"** | **Raised by**: Validator B | **Dismissal Reason**: Same as AC-2 dismissal - visual requirements don't always need programmatic assertions. Can check grid classes or do visual regression testing.

- **"Verbose Dev Notes with redundant context"** | **Raised by**: Validator B | **Dismissal Reason**: Context repetition is intentional for dev agent clarity. Redundancy aids comprehension. Not a functional issue.
