**🐛 Implicit dependency — `mermaid` not in `package.json`:**
- `astro-mermaid@1.3.1` declares `"mermaid": "^10.0.0 || ^11.0.0"` as a required peer dependency.
- The lock file shows `mermaid@11.13.0` installed with `"peer": true` — auto-installed by npm v7+.
- `package.json` has no `mermaid` entry, meaning:
  - The dependency contract is invisible to readers of `package.json`
  - Adding `--legacy-peer-deps` anywhere in CI silently drops it
  - A future `npm install` could resolve to a different minor version without any `package.json` change
- 📍 `docs/package.json` (missing entry), `docs/package-lock.json:5535`
