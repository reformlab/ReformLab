## Findings (Ordered by Severity)

1. **CRITICAL: Path traversal/delete vulnerability via `run_id`.** `ResultStore` trusts raw `run_id` for filesystem path construction; `delete_result("..")` can target parent directories.
2. **CRITICAL: Task 2.4 false-complete.** Delete route says “store and cache” but only deletes store; cache API has no delete.
3. **CRITICAL: Run payload contract broken.** Frontend sends `parameters`, backend consumes `policy`, so scenario policy changes can be silently ignored.
4. **CRITICAL: Task 2.7 false-complete in tests.** No test exercises actual simulation failure path with metadata persistence.
5. **IMPORTANT: Portfolio flow is not implemented end-to-end.** Runner never sends `portfolio_name`; backend metadata `run_kind` stays `"scenario"`.
6. **IMPORTANT: AC-4 detail visualization incomplete.** Required chart/cards reuse is missing.
7. **IMPORTANT: AC-4 export action incomplete.** Buttons render but do nothing from runner context.
8. **IMPORTANT: Timestamp handling bug in result listing.** Invalid timestamp can break whole list sorting.
9. **IMPORTANT: Test suite omits CSV/Parquet success path.** Only 404/409 tested.
10. **IMPORTANT: Export endpoints are memory-heavy.** Full table serialized into in-memory buffers before response.

## Suggested Fixes

1. Add strict `run_id` validation (`UUID v4` regex) in routes and `ResultStore`; reject non-conforming IDs with 422.
2. Add `ResultCache.delete(run_id)` and call it in `DELETE /api/results/{run_id}`.
3. Unify request contract to `policy` (or alias `parameters` in backend with deprecation path), then update frontend callers/types.
4. Wire portfolio execution explicitly: send `portfolio_name`, persist `run_kind="portfolio"`, and set `selectedPortfolioName` from portfolio screen.
5. Implement AC-4 UI reuse in `ResultDetailView` (`DistributionalChart`, `SummaryStatCard`) and wire export callbacks (`exportResultCsv`, `exportResultParquet`) from runner.
6. Harden timestamp parsing: normalize to timezone-aware fallback (`datetime.min.replace(tzinfo=timezone.utc)`) or skip invalid entries.
7. Add missing tests: simulation exception path metadata save, export 200 path with cached data, cache eviction on delete.
8. Replace large `BytesIO` export path with chunked/temp-file streaming for large result sets.
