2. **[HIGH] Import does not normalize JSON parse failures into subsystem errors.**

3. **[HIGH] Provenance runtime type safety is broken.**  
   `population_provenance` / `calibration_provenance` can become list/string/null while typed as `dict[str, Any] | None`.

4. **[HIGH] AC-4 traceability is not enforced.**

5. **[HIGH] “Mutable field isolation” tests are lying about what they verify.**  
   Tests only prove disk content is unchanged after re-import; they do not verify mutation behavior of the already-loaded package object.
