## Findings (Specific)

1. `artifact.path` is concatenated and read without confinement checks. A crafted index entry like `../../secret` can escape package root and be hashed/read.
2. `ImportedPackage` is marked frozen but holds mutable `dict` fields directly. Consumers can mutate `policy`/`scenario_metadata`, violating immutability guarantees and story claims.
3. `import_replication_package()` does not wrap `zipfile.BadZipFile`; wrong exception type escapes domain boundary.
4. `_load_package_from_dir()` does not wrap `PackageIndex.from_json()` failures; `ValueError` leaks instead of `ReplicationPackageError`.
5. `reproduce_from_package()` extracts years/seeds outside guarded execution. Missing `year`, empty table, or non-int seed crashes with raw exceptions.
6. `_compare_panel_tables()` converts integer values to `float`; precision loss for large int IDs/amounts can produce false diffs.
7. `_compare_panel_tables()` repeatedly calls `to_pylist()` on full columns; this is high-memory/high-latency for large panels.
8. Test suite misses critical adversarial cases and overstates immutability coverage; current tests only block attribute reassignment, not nested mutation.

## AC / Task Gap Mapping

- **AC-4 partial:** malformed ZIP and malformed index JSON do not consistently raise `ReplicationPackageError`.
- **AC-2 partial:** reproduction failure handling is incomplete because pre-run config extraction exceptions are unhandled.
- **Task 1.1/6.2 overclaimed:** “frozen immutability” marked complete, but nested mutable fields remain mutable and untested.

- Story file list claims only 3 touched files for 16.2, but provided git diff shows broad unrelated calibration changes in the same working set. This is traceability/scope contamination risk for review and release.

## Suggested Fixes (No edits applied)

1. Add path confinement check: `resolved = (package_dir / artifact.path).resolve(); resolved.is_relative_to(package_dir.resolve())`.
2. Wrap all import-time parse/load failures into `ReplicationPackageError` with artifact context.
3. In `ImportedPackage.__post_init__`, defensive-copy mutable dicts (and optionally normalize `pa.Table` immutability expectations).
4. Validate `year` column existence/non-empty and seed coercion with explicit `ReplicationPackageError`.
5. Compare integer columns as integers; use tolerance only for floating types.
6. Replace `to_pylist()` whole-column comparisons with chunked/vectorized PyArrow comparisons.
7. Add tests for: malformed ZIP, malformed index JSON, negative tolerance, path traversal artifacts, nested dict mutation immutability.
