## Story 9-4 (2026-03-01)

| critical | O(nĂ—g) nested loop in Step 5e builds group entity instances with an outer loop over distinct group IDs and an inner Python loop over all n person rows, calling `.as_py()` per element. For 250k persons Ă— 100k households = 25 billion scalar-boxing operations at ~10M ops/sec â‰ˆ 2+ hours per `compute()` call.
