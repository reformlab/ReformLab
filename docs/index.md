# ReformLab — Project Documentation Index

**Generated:** 2026-03-08

## Start Here

ReformLab is an OpenFisca-first environmental policy analysis platform. It simulates the effects of carbon taxes, subsidies, and other policies on household populations over multiple years. The platform provides three interfaces: a Python API, Jupyter notebooks, and a no-code GUI.

## Getting Started

### For Developers

```bash
# Backend
uv sync --all-extras --dev
uv run pytest tests/ -x --tb=short

# Frontend
cd frontend && npm ci && npm run dev
```

### For AI-Assisted Development

Point your AI tool to this index file (`docs/index.md`) as the entry point. The [Architecture](./architecture.md) document explains all design patterns and conventions. The [Development Guide](./development-guide.md) covers coding standards and quality gates.

### For Analysts

Start with the [quickstart notebook](../notebooks/quickstart.ipynb) or launch the GUI:

```bash
# Start backend
uv run uvicorn reformlab.server.app:create_app --factory --port 8000

# Start frontend
cd frontend && npm run dev
# Open http://localhost:5173
```
