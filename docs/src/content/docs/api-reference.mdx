---
title: API Reference
description: Python API and REST API reference for ReformLab.
---

ReformLab exposes two interfaces: a **Python API** for programmatic use in notebooks and scripts, and a **REST API** (FastAPI) for integration with external tools and the no-code GUI.

A condensed API reference covering both interfaces will be added in Story 19.4, generated from docstrings and OpenAPI schema. For the current OpenAPI spec, run the backend and visit `http://localhost:8000/docs`.
