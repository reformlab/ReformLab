title: Getting Started
description: Run your first ReformLab simulation in four steps.
---

ReformLab runs locally on a standard laptop (16 GB RAM) and requires Python 3.13+ and OpenFisca France.

A step-by-step quickstart guide will be added in Story 19.3, covering: (1) prepare a synthetic population, (2) choose a policy template, (3) configure the computation engine, and (4) run the simulation and inspect results.

Until then, see the project [README](https://github.com/reformlab/reformlab) for installation instructions.
