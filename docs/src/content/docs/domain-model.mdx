---
title: Domain Model
description: Core concepts and objects in the ReformLab domain model.
---

ReformLab is built around six core domain objects: **Population** (the household dataset), **Policy** (the reform being evaluated), **Engine** (the OpenFisca computation backend), **Simulation** (a single policy-run combination), **Results** (year-by-year panel output), and **Indicators** (derived metrics like Gini coefficient or revenue yield).

An interactive diagram and detailed explanations for each object will be added in a future update.
